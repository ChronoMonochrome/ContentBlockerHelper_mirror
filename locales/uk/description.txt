Розширення дозволяє з легкістю блокувати рекламу, відстеження, скрипти і будь-який вміст .

= Можливості

* Блокування вмісту за допомогою підписок (Передплата "Fanboy(Main)" включена за замовчування).
* Блокування обраних елементів кліком миші.
* Блокування вбудованих об'єктів та/або скриптів за допомогою вбудованого блокувальника Opera.
* Приховування елементів за допомогою CSS правил.
* Автоматичне блокування скриптів (замовчуванням вимкнено)
* Можливість імпорту фільтрів блокування AdBlock Plus 1.1 . (правила для блокування вмісту $ третіх - сторін та інше)
* Можливість використання правил блокування при відключеному розширенні (Експорт CSS і додавання правил в список вбудованого блокувальника)

= Використання

== Додавання підписок

Підписка на Fanboy (Main) включена за замовчуванням. Виберіть потрібні Вам підписки на вкладці " Передплати" сторінки налаштувань і натисніть на кнопку " Підписатися " .

== Імпорт списку блокування AdBlock Plus 1.1

Перейдіть у вкладку "Інструменти" і натисніть на кнопку "Імпортувати файл AdBlock Plus 1.1".
Правила блокування будуть сконвертовані і заповнять вкладки " Заблокований вміст " і "Правила CSS". Відкрийте кожну вкладку та натисніть кнопку "Зберегти".

Увага! Процес імпорту може проходити повільно через конвертації правил блокування.

# Натисніть на іконку розширення на панелі браузера.
# Виберіть тип вмісту, який Ви хочете заблокувати.
# Натисніть на елемент, який хочете заблокувати.
# Виберіть правило зі спливаючого списку і натисніть Ok.
# Натисніть на іконку розширення на панелі браузера.
# Натисніть на кнопку "Готово".
# Оновлення сторінки, якщо правила застосувались не відразу.

== Блокування скриптів

Автоматичне блокування скриптів замовчуванням вимкнено. Ви можете включити його на вкладці "Управління"

; Відключено : Автоматичне блокування відключено.
; Скрипти з інших сайтів : Скрипти сайту і з усіх під -доменів будуть завантажуватися.
; Всі зовнішні скрипти : Скрипти сайту будуть завантажено, але скрипти з під -доменів будуть заблоковані.
; Всі скрипти : Будуть заблоковані всі скрипти на сторінці. Якщо Ви хочете дозволити завантаження певних скриптів - додайте їх у білий список.

Натисніть на кнопку "Тимчасово дозволити всі скрипти" оновіть сторінку.

== Додавання правил в список блокованою вмісту для подальшої синхронізації.
Ви можете перенести правила блокування з тимчасового списку в постійний. При перенесенні правил вони будуть вилучені з тимчасового списку.

== Підтримка Opera Mobile Labs
Якщо Ви включите опцію "Завжди додавати правила в список блокованою вмісту ", то Вам більше не знадобиться вручну міняти urlfilter.ini . Досить буде включити синхронізацію правил блокування вмісту.
